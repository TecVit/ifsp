## INTERFACES WEB 2

<p>A avaliação será realizada por meio de três instrumentos: prova, exercícios práticos de fixação e projeto.
Os exercícios práticos consistem na implementação de atividades associadas ao conteúdo apresentado, e são realizados durante as aulas e também como atividade extraclasse.</p>

<p>O projeto será desenvolvido por meio da implementação de software que utilize os conceitos estudados no componente curricular.</p>

A média do semestre será calculada pela seguinte fórmula:

```bash
MS = (PR * 0.40) + (EX * 0.30) + (PJ * 0,30),
na qual:

MS = Média Semestral

PR: Prova prática

EX: Exercícios

PJ: Projeto Os critérios para aprovação e reprovação são especificados na Organização Didática dos Cursos do Integrado do IFSP (Resolução IFSP nº 147/2016).
```
